# Cloud Native Platform 🚀

A production-grade Cloud Native Platform built from scratch to demonstrate modern Platform Engineering practices.

## 🎯 Project Goal

This repository documents the complete journey of building a cloud-native platform similar to what Platform Engineering teams build in production.

The objective is not only to deploy an application but to design, automate, secure, and operate the complete platform using Infrastructure as Code, Kubernetes, GitOps, and CI/CD.
